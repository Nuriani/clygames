crear un proyecto
ng new nombreDelProyecto
ng new nombreDelProyecto --skip-install

dentro del proyecto
yarn install

Creamos un componente
ng g c nombreComponente

Crear un servicio
ng g s nombredelservicio

crear Guard
ng g guard guards/horaAcceso

Instalar bootstrap
npm install bootstrap jquery @popperjs/core

"styles": [

para verificar si tienen instalado el node js
node -v


crear la carpeta node_modules
npm i

correr servidor
ng serve --o --port 4100
